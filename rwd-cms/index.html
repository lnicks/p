Integration

Primary challenge is merging two systems comprised of multiple subsystems

  • CMS (Drupal, Wordpress)

  • Responsive Framework (Bootstrap, Foundation, Webblocks)

Each of application is frequently bundled with more frameworks and languages, providing even more items to be aware of in development

Obstacles to integration

  • Semantics & structures are different

  • Open source development challenges

Semantics & Structures

  • CMSâ€™s come with existing structures (ex: navigation), not always responsive

  • Responsive frameworks work via structures, css, and JavaScript

  • Data manipulation is handled on the backend, responsive design is heavily front end oriented

  • Additional libraries require different versions / levels of support (jQuery, browser degradation)

Open source development challenges

  • Open source CMS communities develop modules which add functionality

  • Modules are yet another piece of software created by users, frequently without the same level of support as core, compounding existing issues

  • Integration of responsive frameworks and elements to outdated or highly rigid structures is difficult
